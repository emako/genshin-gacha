## 功能
 - 读取客户端日志文件中的url
 - 将获取到的6个月历史记录与本地记录合并（1.3版本更新后api只能获取6个月历史记录）
 - 导出抽卡记录为带格式的excel表格
 - 展示抽卡报告 [抽卡报告预览](抽卡报告.md) ，鼠标悬停在数量上即可查看对应星级详情
 - 自动抓包本机api请求
 - 记录历史url，如果可用无需重新抓包
 - 可选清除历史生成的excel表格
 - 可选手动输入url
 - exe版无需fiddler和python环境
